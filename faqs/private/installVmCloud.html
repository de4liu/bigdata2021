Install the MSBA6330 virtual machine on your virtual desktop

In this tutorial you will setup your MSBA virtual machine for use on your virtual desktop.

Once you're in your virtual desktop:

  1. Opening Git CMD
  2. cd c:/
  3. git clone https://github.umn.edu/deliu/vagrant.git vagrantcopy -- to clone the vagrant folder from instrutor's git repository vagrant.
  4. Typing your x500 and password when prompted (credentials get saved to Windows Credential Manager)
  5. cd c:/vagrantcopy
  6. cp -r .git c:/vagrant/.git -- copy the .git folder to c:/vagrant folder which makes this folder linked to instrutor's git repository
  7. rm -rf c:/vagrantcopy -- Remove the temporary git repository
  8. cd /vagrant
  9. vagrant box add MSBA6330 Y:/MSBA6330.box -- create a virtual machine named MSBA6330 using the package file on Y drive.
  10. vagrant up -- initialize and start the virtual machine. Next time, if you need to start the VM, just go to c:/vagrant and do a vagrant up
  11. ssh cloudera@localhost -p 2222 -- ssh into the VM via port 2222. When asked if you will continue to connect, answer "yes". When prompted for password, type "cloudera"
  12. Now you can work from the command line through the SSH window, or start a GUI from VirtualBox ("play" the running VM)
  13. When you need to shutdown the VM, go to the git bash, and at c:/vagrant, type vagrant halt
